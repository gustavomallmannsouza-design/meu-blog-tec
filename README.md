# meu-blog-tec# 🚀 Blog de Curiosidades Tecnológicas

Um blog simples, responsivo e interativo focado em compartilhar curiosidades sobre computação e tecnologia. O projeto utiliza HTML5, CSS3 (com Flexbox) e JavaScript puro para a interação dos botões de curtidas e comentários.

## 📋 Funcionalidades

- **Layout Responsivo:** Adapta-se a diferentes tamanhos de tela usando Flexbox.
- **Sistema de Interação:** Botões com contadores dinâmicos que aumentam e diminuem ao serem clicados.
- **Estruturação Limpa:** Código modular dividido corretamente entre estrutura, estilo e comportamento.

## 📁 Estrutura do Projeto

```text
├── index.html       # Estrutura e conteúdo do blog
├── style.css        # Estilização e design responsivo (Flexbox)
├── script.js        # Lógica de interação dos botões
└── README.md        # Documentação do projeto
```

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica da página (`<header>`, `<main>`, `<article>`).
- **CSS3:** Estilização com foco em Flexbox (`display: flex`, `flex-wrap`, `gap`).
- **JavaScript (ES6):** Manipulação do DOM para interatividade dos botões de engajamento.

## 🚀 Como Executar o Projeto

1. Faça o download ou clone este repositório em sua máquina.
2. Certifique-se de que os arquivos `index.html`, `style.css` e `script.js` estão na mesma pasta.
3. Adicione uma imagem com o nome `imagem-blog.png` no mesmo diretório (ou altere o caminho no HTML).
4. Abra o arquivo `index.html` diretamente em qualquer navegador web (Chrome, Firefox, Edge, etc.).
